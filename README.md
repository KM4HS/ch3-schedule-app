# 일정 관리 앱
Spring boot 실습을 위해 제작한 일정 관리 앱입니다.
아래 목록의 기능을 구현했습니다.
- 일정 생성
- 일정 조회
  - 단건 조회
  - 조건별 다건 조회
  - 페이지 단위 조회
- 일정 수정 및 삭제

- 유저 생성, 조회, 이름 수정
